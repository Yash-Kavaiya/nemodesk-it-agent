# Runbook RB-008: Software Install / Application Errors

Category: software
Symptoms: App won't install, license errors, crashes, or specific error codes.

## Resolution Steps
1. Confirm the user has the entitlement/license assigned.
2. Install via the managed software portal (Company Portal / SCCM) rather than manual download.
3. Clear the app cache or reinstall; run installer as administrator if policy allows.
4. Capture the exact error code and search the vendor knowledge base.
5. Verify OS prerequisites and pending Windows updates.

## Escalation
Licensing or packaging issues -> App-Support. Provide app name, version, and error code.
